# MN-school-progress-project-analysis

This repository contains public-facing documentation and analysis materials for the School Progress Project's study of Minnesota elementary reading achievement.

## Current contents

- `MCA_Data_Dictionary.pdf`  
  Definitions and notes for the variables used in the analysis.

- `MCA_Data_Cleaning_Log.pdf`  
  Summary of data preparation, aggregation, missing-data handling, and analytic sample construction.

- `MCA_reading_analysis_report.pdf`  
  Public-facing report describing the model, diagnostics, results, and interpretation.

- `mn_reading_analysis.Rmd`  
  The main analysis document in R Markdown format. This file is being prepared for clearer public inspection and may not yet be fully reproducible from this repository alone.

## Purpose

The goal of this repository is to support transparency around the School Progress Project's analysis of school-level reading proficiency in Minnesota.

## Current status

This repository currently emphasizes documentation transparency. Full reproducibility may require additional public-ready code, data files, and setup instructions.

## Main analytic focus

The central question is whether district elementary-school structure moderates the relationship between school poverty and reading proficiency.

The primary reporting model examines whether schools in very small districts (1 to 3 elementary schools) show a less steep decline in predicted reading proficiency as poverty rises than schools in districts with 4 or more elementary schools.

This is an evolving public repository. Files and documentation may be refined over time as the project develops.
